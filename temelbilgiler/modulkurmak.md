### Moduller

+ Python modül (paket) yönetim sistemi

```python
sudo apt-get install python-pip   # python2
sudo apt-get install python3-pip  # python3
```

+ Yüklü modülleri listeleme

```python
pip list  # python2
pip3 list # python3
```

+ Paket aramak

```python
pip search paket_ismi   # python2
pip3 search paket_ismi  # python3
```
